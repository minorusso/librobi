# README

### 開発言語
- Ruby2.7.5
- Rails6.0.4.6  

### 使用技術
- Heroku
- devise
- Ajaxを使ったコメント機能
- フォロー機能
- お気に入り機能（favorite）
- rails-admin
#### 指定外
- 楽天ブックスAPI
https://webservice.rakuten.co.jp/documentation/books-total-search

### 実行手順
```
$ git clone https://github.com/minorusso/librobi.git
$ bundle install
$ rails db:create db:migrate
$ rails s
```
